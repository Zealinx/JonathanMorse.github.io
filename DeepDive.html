Deep Dive

A 2D Dungeon Crawler developed in a pre-made custom engine that's set in a mysterious dungeon found deep below the ocean depths. Explore each room of the dungeon, looking for new bullet types and other artifacts to aid you in your quest to reach the bottom.

On this project, I was responsible for creating a system that managed all of the items in the game. I created an inventory system using a linked list, and created a system that allowed the players to equip items and equipment.

Being responsible for the items, I also implemented the behaviors for said items. Some were simple checks to perform actions, such as the magnet allowing the Player to move metal crates and the hammer allowing the player to smash wooden crates, and some were more standalone. The fishing rod, for example, allows the Player to place a Bait object in the direction of their mouse, and if that Bait is deployed, the enemies would target it instead of the Player.

I developed the behaviors for the equipment as well. These were also relatively simple, mainly modifying some Player stats, whereas one slowly heals the Player over time.

Lastly, I crafted all of the UI elements relating to the items. The inventory, the HUD, and the "Item Get" screens.

Overall, for the first game project I've ever made, I feel pretty proud of it. I was able to put the little that I had learned by this point to use and create systems that, though could be improved, still hold up. As a team, we held frequent in-person meetings, mainly just in case we needed to communicate to each other. For the most part, we managed our systems with little overlap. Any problems that arose were mainly due do a lack of experience, both in programming in general as well as working on a team project for the first time.
